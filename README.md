# Student Grade Predictor

A simple Machine Learning project that predicts a student's final score
based on study habits and academic performance.

------------------------------------------------------------------------

## Overview

This project uses **Linear Regression** to estimate a student's final
grade using factors like:

-   Study hours per day\
-   Attendance percentage\
-   Sleep hours\
-   Previous academic score

The goal is to understand how different habits affect performance and
build a basic ML model.

------------------------------------------------------------------------

## Technologies Used

-   Python\
-   NumPy\
-   Pandas\
-   Matplotlib\
-   Scikit-learn

------------------------------------------------------------------------

## How It Works

1.  A synthetic dataset is generated using random values\
2.  A Linear Regression model is trained on the dataset\
3.  The model predicts final scores based on input features\
4.  Model performance is evaluated using Mean Absolute Error\
5.  A simple graph compares actual vs predicted values

------------------------------------------------------------------------

## How to Run

1.  Install required libraries:

    pip install numpy pandas matplotlib scikit-learn

2.  Run the Python file:

    python project.py
